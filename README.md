# Equal Subarray Visualizer

Interactive step-by-step visualization of the "Longest Subarray with Equal 0s and 1s" algorithm using the prefix sum trick.

## Deploy to Vercel (3 steps)

1. Push this folder to a GitHub repo
2. Go to [vercel.com](https://vercel.com) → Import Project → select your repo
3. Click Deploy — Vercel auto-detects Vite, no config needed

## Run Locally

```bash
npm install
npm run dev
```

Then open http://localhost:5173
